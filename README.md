# JavaLibraryProjectL2
Projet Java universitaire dans lequel nous créons un logiciel qui imite un logiciel de gestion de bibliothèque grâce à la bibliothèque graphique Swing.

# Database
Il faut un serveur de base de donnée MySql ou MariaDB en local.
Créer la base de données 'bibliotheque' puis importer 'bibliotheque.sql', par exemple avec PhpMyAdmin.
Bien penser à mettre root comme login et à mettre aucun mot de passe (ou modifier dao.SingleConnection.getInstance() en conséquence).

# Lire le projet
Installer Eclipse : https://www.eclipse.org/downloads/
Installer au minimum le JDK 21 : https://www.oracle.com/fr/java/technologies/downloads/#java21

# Informations
Des abonnements sont automatiquement générés lors de l'import de la base de données.
Un bibliothécaire et son compte associé sont créés au lancement du programme (Identifiant : DoeJ, Mot de passe : Jd12).
Le reste peut être géré par le logiciel.
